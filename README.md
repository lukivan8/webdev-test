# Экзамен по веб-разработке

Онлайн-экзамен на Next.js + Convex с автопроверкой и таблицей результатов в реальном времени.

## Запуск

```bash
# 1. Установить зависимости
npm install

# 2. Настроить Convex (создаст проект и .env.local)
npx convex dev

# 3. В другом терминале запустить Next.js
npm run dev
```

## Страницы

- `/` — страница экзамена (ввод имени, прохождение теста, отправка)
- `/dashboard` — таблица результатов в реальном времени с экспортом в CSV

## Переменные окружения

Файл `.env.local` создаётся автоматически при `npx convex dev`:

```
NEXT_PUBLIC_CONVEX_URL=https://your-project.convex.cloud
```
